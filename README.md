# MapleStory-Campus
메이플스토리 게임에서 필요한 부가적인 기능을 제공합니다.

## 개발 환경
* Java 11
* Gradle 7.6
* Spring Boot 2.7.9

## Dependencies
* Lombok
* Thymeleaf
